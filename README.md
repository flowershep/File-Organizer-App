энэхүү програмыг ажиллуулахаас өмнө танд хэдэн зөвлөгөө та хэлэх хэрэгтэй.
энэхүү програм нь таны сонгосонавтобус эсгүй бол хатуу байр байршил байгаа бүх програм програм болон файлуудыг бүгдийг нь ангилах учраас та өөрийн гэсэнс болпрограмын хавсыг тусад нь салгаж авах хэрэгтэй эс бөгөөс таны тоглоом болон програм чинь хавчин дотор байгаа бүх файлуудыг хураагаад цэгцэлэлж гэсэн үг юм.
тэгэхээр таны тоглоом эсвэл файлууд чинь задарсан учраас та эдгээр програмуудыг дахин ашиглах боломжгүй болно гэсэн үг.
ийм учраас та хэрэгтэй хавтас болон програмуудаа салгаж тусад нь тавьсныг дараа энэхүү програмыг ашиглахыг зөвлөж байна.
хэрвээ энэхүү програм нь танд таалагдах юм бол бидэнтэй холбоо бариарай.
